# QA Check — 품질 검증 스킬

## 절차
1. `npx next build` 빌드 검증
2. TypeScript 타입 에러 확인
3. 기능별 테스트 시나리오 실행
4. 엣지케이스 점검
5. 이슈 보고

## 빌드 검증
```bash
npx next build 2>&1 | tail -20
# 모든 페이지 성공 확인
```

## 주요 테스트 포인트
### 루틴 모드
- 등원/잠자리 모드 전환
- 각 모드별 태스크 목록 정확성
- 모드별 테마 색상/텍스트

### 타이머
- 시간 카운트다운 동작 (매초 감소)
- 오버타임 표시
- 일시정지/재개

### 데이터 호환
- 새로 설치 (빈 localStorage)
- 기존 데이터 (bedtimeTasks 없는 상태)
- 설정 초기화

## 이슈 심각도
- **Critical**: 앱 크래시, 데이터 손실
- **Major**: 기능 미동작, UI 깨짐
- **Minor**: 텍스트 오류, 미세 스타일
