# Plan — 기능 기획 스킬

## 절차
1. 사용자 요청에서 핵심 기능 추출
2. 플랫폼 분류 (웹/모바일/공통)
3. 유저 스토리 작성
4. 수용 기준 정의
5. 작업 의존관계 정리
6. 팀별 작업 분배

## 유저 스토리 템플릿
```
As a [사용자 유형],
I want [기능],
so that [가치/목적].
```

## 수용 기준 템플릿
```
Given [전제조건],
When [동작],
Then [기대 결과].
```
